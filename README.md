## 简介
统一的文件收取,自动归档,记录每次提交的文件信息与提交人信息。

## 项目背景
校园学习生活中会出现以下几个场景:
* 每次碰到上机课的时候,都会遇到收取实验报告。
* 需要收取每个人填写的各种电子表格。
* 类似场景还有不少就不列举了。。。

通常的方式是,通过QQ/微信/邮箱等收取,弊端显而易见,太过于麻烦且不方便整理统计。还占用电脑/手机内存。
为了解决这个问题,此项目应运而生。
